# proyecto_7
Proyecto 7: Creación de una aplicación web para visualizar los datos de vehículos 

Descripción del proyecto

El proyecto consta de la creación de una aplicación web que ayude a visualizar los datos de vehículos vendidos en US respecto al año y el precio de estos mediante gráficos de dispersión e histogramas mediante un botón , además de un gráfico de barras que asocia el precio y el módelo del vehículo mediante un checkbox.


Para realizar esto se creó un entorno virtual para el proyecto donde solo estuvieran los archivos relacionados a este y se instalaron los paquetes de pandas, streamlit y plotly-express.Luego se creo el archivo requeriments.txt donde figuran los principales paquetes del entorno de programación.En paralelo se creo un nuevo repositorio en Git con los archivos README y gitignore.


Se creo el archivo app.py que es la fuente principal de código de la aplicación web, en el que se utilizó plotly-express para crear todos los gráficos asociados al archivo de datos dados.

Conclusiones
Se logró crear una aplicación web básica y funcional, estas habiliades pueden ser aplicadas a todos los campos del análisis de datos en el fututo mostrando bastante potencial
